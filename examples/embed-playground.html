Arguments

  • data-api-base-url: backend base URL used by widget requests.
  • data-chatbot-id: value passed to embed token endpoint.
  • data-title: chat window title.
  • data-target: optional mount selector; widget auto-creates mount if missing.
  • data-max-history-items: max browser-side history retained.
  • data-start-open: open window immediately on load.
  • data-draggable: enable desktop drag behavior.
  • data-launcher-icon: emoji/text or image URL icon.
  • data-locale: fixed locale (`en` or `zh-TW`).
  • data-locale-query-param: query key for locale override.
